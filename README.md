# Algorithmic Graph Theory 
Solving graph problems for MA3052 Teori Graf Algoritmik 2025/2026

# Compilation 
note: Please use linux (or MAC), if you're using Windows, use WSL.

```bash
# 1. Navigate to the project root
cd algorithmic-graph-theory-assignment

# 2. Generate build files
cmake .

# 3. Compile
make

# 4a. Run
./GraphSolver

# 4b. Run with a graph file (default location: tests/)
./GraphSolver graph1.txt
```
